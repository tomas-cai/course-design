# 通用工具参考（Common Tools）

贯穿全流程的工具和方法。Codex 在对应步骤中按需读取。

## 布卢姆行为动词手册

### 完整动词表

| 层级 | 常用动词 | PM 课程推荐动词 | 不推荐（太模糊） |
|:---|:---|:---|:---|
| **记忆** | 列出、识别、命名、复述、定义 | 列举、识别 | 了解、熟悉 |
| **理解** | 解释、总结、举例、分类、推断 | 解释、总结、类比 | 理解、掌握 |
| **应用** | 执行、实施、使用、演示、操作 | 使用、制作、演示 | 运用 |
| **分析** | 区分、组织、归因、比较、拆解 | 比较、分析、拆解 | 分析清楚 |
| **评价** | 评估、判断、辩护、推荐、排序 | 评估、判断、推荐 | 评价 |
| **创造** | 设计、构建、规划、开发、制定 | 设计、规划、提出方案 | 创新 |

### 动词选择原则

1. 动词必须**可观测**——别人能看到学员完成了这个动作
2. 动词必须**可测评**——有人能用评分标准评价完成质量
3. 优先用**具体动词**避免**抽象动词**
   - ❌ "学员能理解用户需求"（理解不可直接观测）
   - ✅ "学员能**列出**至少 5 种用户需求分析方法，并**选择**一种**设计**一份访谈提纲"（三个具体行为可观测）

## 敏捷验证节点模板

### P1 验证（分析→设计之间）

验证问题：
- 学员是否真的有这个痛点？痛到什么程度？
- 学员对课程主题的预期是什么？
- 我们设计的解决方案方向是否对？

操作：
- 找 3 个目标学员做 30 分钟访谈
- 不需要任何教学内容，只需要一个课程概述（1 页纸）

### P2 验证（设计→开发之间）

验证问题：
- 课程结构是否合理？模块之间逻辑是否自然？
- 体验旅程是否有效？情绪曲线是否合理？
- 第一个"惊叫点"设计是否有效？

操作：
- 准备 1 小时的最小原型课程（PPT + 一个关键练习）
- 找 2-3 人做一次试讲
- 收集反馈后调整设计

### P3 验证（开发中期~50%）

验证问题：
- 前 20% 的内容是否达到了预期学习效果？
- 节奏和认知负荷是否合理？
- 是否有需要大规模调整的地方？

操作：
- 用已开发的前 20% 内容做一次完整试课
- 3-5 名目标学员
- 收集反馈，决定剩余 50% 是否需要调整方向

## 课程开发质量检查总清单

### 分析阶段检查
- [ ] 需求来源是否明确？
- [ ] 是否完成了痛点验证（不只有一个需求方确认）？
- [ ] 是否识别了不适合培训解决的问题？
- [ ] 学习者画像是否包含了情绪起点分析？
- [ ] 能力差距诊断是否基于布卢姆分层？

### 设计阶段检查
- [ ] 每条学习目标是否使用"行为动词 + 可观测成果"格式？
- [ ] 目标层级是否匹配业务期望？
- [ ] 是否设计了情绪曲线？不是只有内容大纲？
- [ ] 每条目标是否有对应的内容和测评？
- [ ] 是否有至少一个"惊叫点"设计？
- [ ] 是否有认知负荷管理策略？

### 开发阶段检查
- [ ] 四侧材料是否全部到位？（学员/讲师/实操/测评）
- [ ] 目标-内容-测评是否对齐？
- [ ] 实操案例是否有明确决策点？
- [ ] 测评形式是否匹配布卢姆层级？
- [ ] 视觉是否降低了认知负荷？
- [ ] 是否执行了至少一次敏捷验证？（P1/P2/P3）？

### 实施阶段检查
- [ ] 是否做了前置测评？
- [ ] 是否需要分层教学？
- [ ] 是否有随堂形成性评估计划？
- [ ] 讲师引导手册是否包含时间分配和关键话术？

### 评估阶段检查
- [ ] 是否覆盖了柯氏 L1-L4？
- [ ] 是否对每项目标做了达成度分析？
- [ ] 是否复验了最初的"痛点假设"？
- [ ] 是否输出了优先级明确的迭代计划？

## 柯氏四级评估检查表

### L1 反应层
- [ ] 收集整体满意度评分
- [ ] 收集 NPS（净推荐值）
- [ ] 收集痛点解决度反馈
- [ ] 收集惊叫点命中率反馈
- [ ] 收集爽点体验反馈

### L2 学习层
- [ ] 执行后测
- [ ] 对比前置测评数据
- [ ] 每项目标的达成率计算
- [ ] 实操考核通过率

### L3 行为层
- [ ] 设定课后跟进时间点（1 个月 / 3 个月）
- [ ] 收集工作中的应用频率和场景
- [ ] 识别应用中的障碍
- [ ] 提供补充支持（如需要）

### L4 业务层
- [ ] 确定可量化的业务指标
- [ ] 对比培训前后的数据变化
- [ ] 收集上级的主观评价
- [ ] 计算培训 ROI

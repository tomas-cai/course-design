# 培训需求发现 Skill

## 核心理念

不要靠直觉选课程方向——用结构化方法从信号中发现机会。

本 Skill 遵循"**先发散、后收敛**"的原则：先不加判断地收集信号，再通过结构化分析收敛到具体的课程假设。

## 工作流程

### Step 1：信号采集（发散）

引导用户从三个维度列出所有观察到的信号。**不留判断，只列事实。**

**市场信号**——市场上发生了什么变化？
| 维度 | 引导问题 |
|:---|:---|
| 技术趋势 | 有什么新技术正在改变工作方式？ |
| 竞争者动态 | 同行/竞品在推什么课程？ |
| 招聘市场 | 什么岗位需求激增？什么技能被频繁提及？ |

**用户信号**——从目标人群身上看到了什么？
| 维度 | 引导问题 |
|:---|:---|
| 用户痛点 | 他们在什么场景下说"好难""好烦""不知道怎么办"？ |
| 自学行为 | 他们在自学什么？搜什么关键词？ |
| 职场焦虑 | 他们在焦虑什么？（怕被替代、怕选错方向） |

**组织信号**——团队/组织在被什么困扰？
| 维度 | 引导问题 |
|:---|:---|
| 能力断层 | 团队需要什么能力但招不到/培养不出？ |
| 工具变革 | 团队引入了新工具但用不起来？ |

> 详细方法见 `references/discovery-methodology.md` → Step 1

### Step 2：信号聚类 → 痛点提取（收敛）

**信号分组**：把信号按"可能属于同一个培训方向"分组。每个组至少 3 条信号才能作为候选方向。

| 候选方向 | 相关信号 | 初步判断 |
|:---|:---|:---|
| 方向 A | 信号 1, 3, 5 | 可能是培训机会 |
| 方向 B | 信号 2, 4 | 仅 2 条信号，需补充 |

**痛点分级**：
- **L3 紧迫痛点**：用户已在找解决方案，不解决有明确损失
- **L2 成长痛点**：用户知道该提升但还没紧迫感
- **L1 兴趣痛点**：学了也好但不急

**区分"培训能解决的" vs "培训不能解决的"**

| 可培训 | 不可培训 |
|:---|:---|
| 知识/技能缺失 | 组织/流程问题 |
| 方法论不清晰 | 工具采购决策 |
| 判断力不足 | 薪酬/激励问题 |

> 详细方法见 `references/discovery-methodology.md` → Step 2

### Step 3：商机评估

对每个 L2 以上的候选方向做五维评分：

| 维度 | 问题 | 1-5 |
|:---|:---|:---:|
| 市场容量 | 潜在学员有多少？ | |
| 付费意愿 | 学员或公司愿意付多少？ | |
| 竞争强度 | 市场上有多少同类课程？（低竞争=高分） | |
| 内容稳定性 | 这个方向的内容多久过时？ | |
| 我方优势 | 我们有什么独特积累？ | |
| **总分** | **加权平均** | **/ 5** |

> 总分 < 3.0 建议暂缓。

> 详细方法见 `references/discovery-methodology.md` → Step 3

### Step 4：课程假设形成

对通过商机评估的方向，形成一页纸课程假设：

| 字段 | 内容 |
|:---|:---|
| 暂定课程名 | |
| 一句话价值主张 | |
| 目标人群（初步） | |
| 核心痛点（初步） | |
| 课程类型 | 认知课 / 技能课 / 思维课 |
| 交付形式 | |
| 为什么是我们做 | |
| 不做的风险 | |

> 详细方法见 `references/discovery-methodology.md` → Step 4

## 产出

写入 `00-培训需求spec.md` 到用户指定的课程项目目录。

文件格式：
```markdown
# 培训需求 Spec：[暂定主题]

## 1. 信号收集摘要
- 市场信号
- 用户信号
- 组织信号

## 2. 候选方向列表
方向 | 信号数 | 痛点等级 | 优先级

## 3. 选定方向商机评估
五维评分表 + 总分

## 4. 课程假设
课程名、价值主张、目标人群、核心痛点...
```

## 跨 Skill 协作

本 Skill 的产出物可直接输入到 **course-design Skill** 作为 Phase 01 的起点：

| 本 Skill 产出 | 对应的 course-design 输入 |
|:---|:---|
| `00-培训需求spec.md` | Phase 01 分析阶段的"前置输入" |

产出的 `00-*.md` 文件可存放在任意课程项目目录下，course-design 会自动检测并读取。
